PROJECTS

SMART FACE

FULL STACK APP

This web app will find the faces on an image URL and draw boxes around each face in the picture. The app has full sign in and register functionality to build a user's profile. The front end of this web app was built using the React framework. The server was built using Node and Express. Profiles and login information are stored in a PostgreSQL database and may be updated and changed. JSON Web Tokens and Redis are used for session management. The back end uses the Docker platform for easier development.

NASA PICTURES

FRONT END APP

This application was built with the React framework in conjunction with Redux to manage the state. This site will load ten random pictures with descriptions from the NASA API. A seachbox enables you to search for keywords in the pictures' title, description, and copyright. Pictures may be added to a favorites page that is stored in local storage.

MS PAINT CLONE

JAVASCRIPT PROJECT

This website mimics the functions of Microsoft Paint. The program was built using Javascript and canvas.

STAR WARS MOVIE SEARCH

FRONT END APP

This web app will show a searchable list of ten Star Wars movies that are pulled from the star wars API. The React framework was used with Redux in this app to manage state. Redux-thunk and redux-logger middleware was used for developement. JEST was used for testing in this application.
